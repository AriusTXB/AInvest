# Product Requirements Document (Inferred)

## Product Name
AInvest Terminal

## Purpose
An automated equity research platform that democratizes access to institutional-grade stock analysis by combining technical, social, and fundamental signals into a single "Investment Memo".

## Core Features
1. **Real-time Market Data**: Price, Volume, and Technical Indicators (RSI, MACD, SMA).
2. **AI Analysis**: Sentiment analysis of news headlines.
3. **Social Signals**: Aggregation of social media sentiment (Mocked/Planned).
4. **Investment Recommendation**: Algorithmically generated BUY/SELL/HOLD signals.
5. **Dashboard**: Interactive terminal interface for investors.

## User Personas
- **Retail Investor**: Wants quick, data-backed summaries without spending hours analyzing charts.
- **Trader**: Needs rapid signal validation from multiple sources.

## Future Roadmap (Implied)
- **Vision Integration**: PDF/Chart parsing via `vision` module.
- **Real Social Integration**: Live scraping instead of mock data.
- **Advanced NLP**: Full article summarization.
